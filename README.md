# Taiyara.closet - Personalized Shoe Experience

A luxury minimalist web application for personalized shoe shopping experience built with Vue.js and Material UI.

## Features
- Product selection grid with smooth transitions
- Interactive shoe customizer
- Measurement input form with validation
- Luxury minimalist design aesthetic

## Technology Stack
- Vue.js 3 with Composition API
- Vuetify (Material UI for Vue)
- Vue Router for navigation
- Pinia for state management
